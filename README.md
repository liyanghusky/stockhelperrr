# Stock Five

Windows-friendly local stock watchlist prototype with five visual styles.

Open `outputs/open-stock-five.bat` on Windows, or open `outputs/stock-watch-five-versions.html` directly in a browser.

Features:

- Add and remove stock symbols
- Switch ranges: 1D, 1W, 1M, 3M, 1Y
- Five themes: Trader, Terminal, Newspaper, Glass, Pastel
- Offline deterministic demo data, no API key required
